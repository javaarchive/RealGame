Welcome to the home of REALGAME

Realgame is a game simulating a real world. You can host your own server here by running the python server script with your hostname on port 80 To start hosting use wget https://github.com/javaarchive/Chatserver/raw/master/server.py then open and edit the varibles ip or hostname and port To open in a linux command line try vi server.py or nano server.py
